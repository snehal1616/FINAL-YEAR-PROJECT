# 🎨 CUROVIA - New Aesthetic Logo Design

## ✨ Logo Overview

The new CUROVIA logo features a modern, premium aesthetic with smooth gradients and elegant animations.

### Visual Design:
```
┌──────────────────────┐
│  ┌──────┐            │
│  │  Cv  │  CUROVIA   │
│  └──────┘            │
│      🟢              │
└──────────────────────┘
```

## 🎨 Design Specifications

### Icon Container:
- **Size:** 48x48px (larger, more prominent)
- **Shape:** Rounded square (14px border-radius)
- **Background:** Triple gradient
  - Cyan (#06b6d4) → Blue (#3b82f6) → Purple (#8b5cf6)
- **Shadow:** Multi-layer depth effect
  - Primary: `0 8px 24px rgba(59, 130, 246, 0.4)`
  - Ring: `0 0 0 3px rgba(59, 130, 246, 0.1)`

### Logo Text:
- **Content:** "Cv" (Curovia abbreviated)
- **Font:** Inter (modern, clean)
- **Weight:** 900 (extra bold)
- **Size:** 1.4rem
- **Color:** White
- **Letter Spacing:** -0.05em (tight, modern)

### Status Badge:
- **Size:** 10x10px
- **Position:** Top-right corner (-3px, -3px)
- **Background:** Green gradient (#10b981 → #34d399)
- **Border:** 2.5px solid white
- **Shadow:** `0 2px 8px rgba(16, 185, 129, 0.6)`
- **Animation:** Pulsing effect

## ✨ Animations

### 1. Shimmer Effect:
```css
- Diagonal light sweep across the icon
- Duration: 3 seconds
- Infinite loop
- Creates premium, polished look
```

### 2. Pulse Effect (Badge):
```css
- Gentle scale animation
- Duration: 2 seconds
- Indicates "active" status
- Smooth ease-in-out
```
